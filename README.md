## Prerequisites

1. Install [RVM](https://rvm.io)
1. Install ruby-2.2.3 (using RVM)
1. Clone the repo

## Getting Started

1. Run `bundle install`
1. Run `rails s` to start the server
1. Open http://localhost:3000 in a browser

## Common Tasks

- `rails s` → runs the server
- `guard` → enables LiveReload
